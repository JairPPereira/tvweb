CSS significa Cascading Style Sheets e é uma linguagem de estilos usada para criar sites.

A grande vantagem do CSS na criação de sites, é que podemos fazer a separação do conteúdo e do formato (estilo).

Geralmente colocamos nosso código CSS em um arquivo de extensão .css e no documento HTML colocamos um link para este arquivo, que é conhecido como folha de estilos.

Um bom site para aprender CSS (além de HTML e HTML5) é o HTML Progressivo: www.htmlprogressivo.net
O Curso HTML Progressivo é online, totalmente gratuito e super completo.
