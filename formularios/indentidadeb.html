Identidades básicas

 

Definições:

  • a : Número real a
  • b : Número real b
  • c : Número real c

Identidade aditiva

a + 0 = a

Inverso aditivo

a + (-a) = 0

Associativo de Adição

(a + b) + c = a + (b + c)

Comutativo de adição

a + b = b + a

Subtração

a - b = a + (-b)

Identidade Multiplicativa

a * 1 = a

Inverso Multiplicativo

a * (1/a) = 1 (se a não for 0)

Multiplicação vezes 0

a * 0 = 0

Associativo de Multiplicação

(a * b) * c = a * (b * c)

Comutativa de multiplicação

a * b = b * a

Lei Distributiva

a(b + c) = ab + ac

Definição de Divisão

a / b = a(1/b)
