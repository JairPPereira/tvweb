Círculo ○

Um círculo é uma forma básica que consiste nos pontos a uma determinada distância de um determinado ponto em um plano.

Definições:

  • r: Raio
  • d: Diâmetro
  • U: Circunferência
  • A: Área

Circunferência de um círculo

A circunferência U é a distância em torno de uma curva fechada.

\( {\color{blue} {U}} = 2 \times \pi \times {\color{red} {r} } \)
U = Circumference
r = Radius

\( {\color{red} {r}} = \frac{ {\color{blue} {U} } }{ 2 \times \pi } \)
r = Radius
U = Circumference

\( {\color{blue} {U}} = \pi \times {\color{red} {d} } \)
U = Circumference
d = Diameter

\( {\color{red} {d}} = \frac{ {\color{blue} {U} } }{ \pi } \)
d = Diameter
U = Circumference

Área de um círculo

A região dentro de um círculo.

\( {\color{OliveGreen} {A}} = \pi \times {\color{red} {r} }^{2} \)
A = Area
r = Radius

\( {\color{red} {r}} = \sqrt{ \frac{\color{OliveGreen} {A}}{\pi} } \)
r = Radius
A = Area

\( {\color{OliveGreen} {A}} = \frac{\pi}{4} \times {\color{red} {d} }^{2} \)
