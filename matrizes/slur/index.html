Solução de sistemas de equações lineares


Solução de sistemas de equações lineares é encontrar o desconhecido variáveis incluídas na equação, ao fazer uma pesquisa de qual o sistema entra em igualdade.

Resolver o sistema de equações lineares pode ser de várias maneiras, por exemplo, o método de Cramer método ou Gause ou de outras formas. Usando o nosso serviço, você será capaz de graça no modo on-line para obter a solução com passo-a-passo as ações e explicações. Nossa a calculadora é também útil se você precisa testar executadas de forma independente do cálculo.

Nosso serviço on-line nos permite resolver sistemas de equações algébricas lineares de várias maneiras:

  • pelo método de Cramer (regra de Cramer)
  • método da matriz inversa
  • pelo método Gauss-Montante (o algoritmo Bareys)
  • pelo método de Gauss (método de eliminação sequencial de variáveis)
  • pelo método de Gauss-Jordan (o método de eliminar completamente as incógnitas)

Nesse caso, o serviço fornece uma seqüência de soluções, não apenas a resposta.

Além disso, você pode verificar o sistema de equações para compatibilidade.

  • Use os sinais + e - para especificar o número de variáveis necessárias na equação. Se a sua equação não incluir quaisquer incógnitas, então deixe os campos em branco (em branco).
  • Nas células, especifique os coeficientes (valores) para incógnitas. Se os dados iniciais indicar o valor x1, x2 e assim por diante, na célula antes das incógnitas especificadas, especifique o valor 1.
  • Os valores para incógnitas podem ser:
    • Inteiros: 7, -3, 0
    • frações decimais (finitas e periódicas): 7/8, 6.13, -1.3(56), 1.2e-4
    • Expressões aritméticas: 1/2+3*(6-4), (6-y)/x^3, 2^0.5
  • Em seguida, clique no botão com o nome da operação matemática necessária.
  • Os valores nos resultados da solução podem ser arrastados com o mouse para o campo de dados de origem.
