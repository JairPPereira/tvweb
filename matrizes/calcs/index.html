  • Como usar

Usando a nossa calculadora de matriz, você será capaz de fazer os cálculos com matrizes, obtendo o resultado desejado de uma resposta detalhada a seqüência de decisões. Matriz calculadora permite a operação com uma única matriz ou resolver expressões complexas com várias matrizes.

Preencha os campos para os elementos da matriz e clique no botão correspondente.

  • Use as teclas "+" e "-" para selecionar o tamanho da matriz desejada. Se você precisar de uma matriz não quadrada, basta deixar as células desnecessárias vazias.
  • Especifique o valor dos elementos da matriz nas células. Os valores podem ser:
    • Inteiros: 7, -3, 0
    • frações decimais (finitas e periódicas): 7/8, 6.13, -1.3(56), 1.2e-4
    • Expressões aritméticas: 1/2+3*(6-4), (6-y)/x^3, 2^0.5
  • Clique no botão com o nome da operação matemática necessária ou no modo manual, digite a expressão matemática em um campo especial.
  • Os valores nos resultados da solução podem ser arrastados com o mouse para diferentes campos. Por exemplo, a matriz resultante pode ser arrastada para o campo de dados de origem, para maior resolução.
