Esta calculadora ajudará você a aprender como encontrar o determinante da matriz de várias maneiras:

  • usando o método de Gauss
  • Usando a regra dos triângulos
  • pela Regra de Sarrius
  • usando a fórmula de Leibniz
  • pelo método de Gauss-Montante (o algoritmo de Bareys)

Nosso serviço não só permite que você obtenha um matador, mas também fornece uma sequência de soluções com comentários e explicações on-line, gratuitamente. Além disso, uma calculadora pode ser útil para verificar a exatidão da solução auto-executada.

Preencha os campos para os elementos da matriz e clique no botão correspondente.

  • Use + e - para selecionar o tamanho da matriz desejado. Se você precisar de uma matriz não quadrada, deixe as células desnecessárias vazias.
  • Digite o valor dos elementos da matriz nas células. Os valores podem ser:
    • inteiros: 7, -3, 0
    • frações decimais (finitas e periódicas): 7/8, 6.13, -1.3(56), 1.2e-4
    • expressões aritméticas: 1/2+3*(6-4), (6-y)/x^3, 2^0.5
  • Clique no botão com o nome da operação matemática requerida.
  • Valores nos resultados da solução podem ser arrastados com o mouse para diferentes campos. Por exemplo, a matriz resultante pode arrastar-se os dados brutos para outras soluções.
